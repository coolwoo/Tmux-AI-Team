# PM 状态面板

查看所有槽位的当前状态。

## 执行步骤

使用 Bash 工具执行：

```bash
pm-status
```

## 输出说明

| 图标 | 状态 | 说明 |
|------|------|------|
| ⚪ | idle | 空闲，可分配任务 |
| 🟢 | working | 工作中 |
| ✅ | done | 已完成 |
| 🔴 | error | 出错 |
| 🟡 | blocked | 被阻塞 |
